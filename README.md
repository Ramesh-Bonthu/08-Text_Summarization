🦜 LangChain: Text Summarization App

A simple and interactive Streamlit-based web application that summarizes content from any website URL using LangChain and Groq LLM (LLaMA 3.1).

🚀 Features
🔑 Secure API key input (Groq API)
🌐 Accepts any valid website URL
📄 Extracts content using UnstructuredURLLoader
🤖 Generates concise summaries using LangChain
⚡ Fast inference with Groq LLM
🎯 Clean and minimal UI with Streamlit

🛠️ Tech Stack
Frontend: Streamlit
Backend: Python
LLM: Groq (LLaMA 3.1 8B Instant)
Framework: LangChain
Loader: UnstructuredURLLoader

📂 Project Structure
├── app.py                  # Main Streamlit app
├── requirements.txt        # Dependencies
├── .env                    # API keys (not pushed to GitHub)
├── TextSummarization.ipynb # Experimentation notebook
